# TimeProject

## Nedir?
TimeProject, belirlediğin süre dolduğunda otomatik olarak komut çalıştıran zamana dayalı bir terminal uygulamasıdır.  
Geri sayaç başlar, süre bittiğinde komut çalışır.

## Süre Formatı
1g 15s 12dk 50sn
**Anlamı:** 1 gün 15 saat 12 dakika 50 saniye

## Hazır Komutlar
- Bilgisayarı kapat  
- Yeniden başlat vb.

## Özellikler
- Zamana dayalı komut çalıştırma  
- Geri sayaç sistemi  
- Otomatik güncelleme desteği  
- Terminal üzerinden kullanım
